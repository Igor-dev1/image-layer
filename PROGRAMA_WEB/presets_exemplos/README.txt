  PRESETS DE EXEMPLO - PROCESSADOR DE IMAGENS EM LOTE
═══════════════════════════════════════════════════════════

Esta pasta contém presets pré-configurados para uso comum.

COMO USAR:
----------
1. Abra o programa principal
2. Clique em "📂 Carregar Preset"
3. Selecione um dos arquivos .json desta pasta
4. Todas as configurações serão carregadas automaticamente

PRESETS DISPONÍVEIS:
-------------------

1️⃣  1_qualidade_maxima_webp.json
   📝 Descrição: Conversão para WEBP com qualidade máxima (lossless)
   🎯 Uso: Quando você precisa da melhor qualidade possível
   ⚙️ Configurações:
      - Formato: WEBP
      - Qualidade: 100% (sem perda)
      - Sem texto

2️⃣  2_badge_promocao.json
   📝 Descrição: Adiciona badge "PROMOÇÃO" em vermelho/amarelo
   🎯 Uso: Produtos em promoção
   ⚙️ Configurações:
      - Formato: WEBP (95%)
      - Texto: "PROMOÇÃO"
      - Cor: Amarelo sobre fundo vermelho
      - Posição: Superior direita
      - Sufixo: "_promo"

3️⃣  3_entrega_imediata.json
   📝 Descrição: Badge "ENTREGA IMEDIATA" em verde
   🎯 Uso: Produtos digitais com entrega instantânea
   ⚙️ Configurações:
      - Formato: WEBP (95%)
      - Texto: "ENTREGA IMEDIATA"
      - Cor: Verde sobre fundo preto
      - Posição: Superior direita

4️⃣  4_todas_dlcs.json
   📝 Descrição: Badge "TODAS AS DLCs" em dourado
   🎯 Uso: Jogos que incluem todas as DLCs
   ⚙️ Configurações:
      - Formato: WEBP (95%)
      - Texto: "TODAS AS DLCs"
      - Cor: Dourado sobre fundo escuro
      - Posição: Inferior direita

5️⃣  5_conversao_png_para_webp.json
   📝 Descrição: Conversão simples PNG → WEBP
   🎯 Uso: Reduzir tamanho de arquivos mantendo qualidade
   ⚙️ Configurações:
      - Formato: WEBP (95%)
      - Sem texto
      - Sem renomeação

6️⃣  6_otimizacao_web.json
   📝 Descrição: Otimização para uso na web
   🎯 Uso: Imagens para sites, lojas online, etc.
   ⚙️ Configurações:
      - Formato: WEBP (85%)
      - Prefixo: "web_"
      - Sem texto
      - Tamanho reduzido mantendo boa qualidade

═══════════════════════════════════════════════════════════

DICAS:
------
✅ Você pode editar estes presets com qualquer editor de texto
✅ Crie seus próprios presets e salve para reutilizar
✅ Combine presets com overlays diferentes para resultados variados
✅ Qualidade 95% é quase indistinguível de 100% com arquivos menores

PERSONALIZAÇÃO:
--------------
Para criar seu próprio preset:
1. Configure tudo no programa
2. Clique em "💾 Salvar Preset"
3. Dê um nome descritivo
4. Salve em uma pasta organizada

VALORES DE QUALIDADE:
--------------------
100% = Lossless (sem perda) - Maior tamanho
95%  = Imperceptível - Recomendado para produtos
85%  = Alta qualidade - Recomendado para web
75%  = Boa qualidade - Para visualizações
60%  = Qualidade média - Para thumbnails
